# 1s

## Problem Statement
- given a linked list consisting of 0s, 1s, and 2s, sort the linked list. (This file represents the 1s segment concept of the Sort 0s, 1s, and 2s algorithm).

## Approach / Intuition
- when sorting a linked list containing 0s, 1s, and 2s, we can count the occurrences of each number in the first pass. In the second pass, we simply update the values of the nodes, replacing the first $C_0$ nodes with 0s, the next $C_1$ nodes with 1s, and the remaining nodes with 2s. This is an alternative to the [[Dummy Nodes]] manipulation technique.

## Time & Space Complexity
- **[[time Complexity]]:** $O(N)$ with two passes
- **[[space Complexity]]:** $O(1)$

## Sample Code
```cpp
struct ListNode {
    int val;
    ListNode *next;
    ListNode(int x) : val(x), next(nullptr) {}
};

ListNode* sortList(ListNode *head) {
    int count[3] = {0, 0, 0};
    ListNode *ptr = head;
    
    while (ptr != nullptr) {
        count[ptr->val]++;
        ptr = ptr->next;
    }
    
    ptr = head;
    int i = 0;
    
    while (ptr != nullptr) {
        if (count[i] == 0) {
            i++;
        } else {
            ptr->val = i;
            count[i]--;
            ptr = ptr->next;
        }
    }
    
    return head;
}
```
